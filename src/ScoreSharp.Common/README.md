# ScoreSharp.Common

## 用途

供 ScoreSharp.API 、 ScoreSharp.Batch 、 ScoreSharp.Middleware 、 ScoreSharp.PaperMiddleware 共用

## 資料夾說明

| 資料夾     | 說明                         |
| ---------- | ---------------------------- |
| Adapters   | 第三方 API HttpClient 適配器 |
| Attributes | 自定義驗證屬性               |
| Constant   | 常數定義                     |
| Enums      | 列舉定義                     |
| Extensions | 擴展方法                     |
| Helpers    | 工具類別                     |
