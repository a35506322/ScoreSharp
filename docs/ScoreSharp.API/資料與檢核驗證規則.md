## 檢驗清單

### 注意事項

1. 狀態表 **Reviewer_ApplyCreditCardInfoHandle 注意要考量多個卡片狀態問題**
2. 網路件與紙本件定義，查看**Reviewer_ApplyCreditCardInfoMain** 的欄位 Source，網路件等於 Ecard 或 APP，紙本件等於 紙本
3. 當執行 **完成月收入確認** / **權限內-核卡作業** / **權限外-排入核卡** 時，需要檢驗資料驗證

### **Reviewer_FinanceCheckInfo 檢核**

1. 929 查詢
   - 在 **Reviewer_FinanceCheckInfo 資料表**
   - 正附卡人必須檢驗否則不能核退補撤
   - 完成月收入確認 / 申請核卡中 / 核卡作業中，檢核日期須為當天
   - 申請核卡中 / 核卡作業中命中不能核卡
2. 姓名檢核
   - 在 **Reviewer_FinanceCheckInfo 資料表**
   - 正附卡人必須檢驗否則不能核退補撤
3. 關注名單 1 + 2
   - 在 **Reviewer_FinanceCheckInfo 資料表**
   - 正附卡人必須檢驗否則不能核退補撤
   - 完成月收入確認 / 申請核卡中 / 核卡作業中，檢核日期須為當天
   - 申請核卡中 / 核卡作業中命中不能核卡
   - 關注名單 1 可核卡
   - 關注名單 2 只有國旅卡能核卡
4. KYC
   - 在 **Reviewer_FinanceCheckInfo 資料表 + 在 Reviewer_ApplyCreditCardInfoMain 資料表**
   - 正卡人必須檢驗否則不能核退補撤
   - 如果為 H + M 需要做加強審核執行，加強審核執行狀態需要核准

### **Reviewer_BankTrace 檢核**

1. 行內 IP 相同
   - 在 **Reviewer_BankTrace 資料表**
   - 網路件 正卡人 IP 必須檢驗
   - 只要有檢核命中，就必須確認是否異常及撰寫紀錄否則不能**核卡作業/排入核卡/完成月收入確認**
2. 相同 IP 比對
   - 在 **Reviewer_BankTrace 資料表**
   - 網路件 正卡人 IP 必須檢驗
   - 只要有檢核命中，就必須確認是否異常及撰寫紀錄否則不能**核卡作業/排入核卡/完成月收入確認**
3. 電子信箱是否相同

   - 在 **Reviewer_BankTrace 資料表**
   - 網路件正卡人電子信箱檢核規則：
     - 如果 Email 未填寫，不需要檢核
     - 如果 Email 有填寫，必須檢核
     - 如果帳單類型選擇「電子帳單」，Email 為必填，因此必須檢核
   - 只要有檢核命中，就必須確認是否異常及撰寫紀錄否則不能**核卡作業/排入核卡/完成月收入確認**

4. 電話是否相同

   - 在 **Reviewer_BankTrace 資料表**
   - 網路件正卡人電話檢核規則：
     - 如果 Mobile 未填寫，不需要檢核
     - 如果 Mobile 有填寫，必須檢核
     - 如果帳單類型選擇「簡訊帳單」，Mobile 為必填，因此必須檢核
   - 只要有檢核命中，就必須確認是否異常及撰寫紀錄否則不能**核卡作業/排入核卡/完成月收入確認**

5. 行內 Email 相同

   - 在 **Reviewer_BankTrace 資料表**
   - 網路件 / 紙本件正卡人 Email 檢核規則：
     - 如果 Email 未填寫，不需要檢核
     - 如果 Email 有填寫，必須檢核
     - 如果帳單類型選擇「電子帳單」，Email 為必填，因此必須檢核
     - **只要帳單類型為電子帳單，就必須驗證**
   - 只要有檢核命中，就必須確認是否異常及撰寫紀錄否則不能**核卡作業/排入核卡/完成月收入確認**
   - 當案件變更動作包含「核卡作業」/「排入核卡」/「完成月收入確認」時，若銀行內 Email 回復異常，不能執行該動作

6. 行內手機相同
   - 在 **Reviewer_BankTrace 資料表**
   - 網路件 / 紙本件正卡人電話檢核規則：
     - 如果 Mobile 未填寫，不需要檢核
     - 如果 Mobile 有填寫，必須檢核
     - 如果帳單類型選擇「簡訊帳單」，Mobile 為必填，因此必須檢核
   - 只要有檢核命中，就必須確認是否異常及撰寫紀錄否則不能**核卡作業/排入核卡/完成月收入確認**
   - 當案件變更動作包含「核卡作業」/「排入核卡」/「完成月收入確認」時，若銀行內手機回復異常，不能執行該動作

### 資料欄位驗證

1. 正卡人 **Reviewer_ApplyCreditCardInfoMain**
2. 附卡人 **Reviewer_ApplyCreditCardInfoSupplementary**
3. 身分證字號 ⇒ 必填，使用 TWID 標籤
4. EMAIL 格式 ⇒ 不必填，符合 EMAIL 格式
5. 生日 ⇒ 必填，民國格式 yyyMMdd
6. 身分證發證日期 ⇒ 不必填，民國格式 yyyMMdd
7. 是否永久居留 ⇒ 完成月收入確認 / 排入核卡 / 核卡作業，身分證字號第二碼不是 1 或 2 需必填，填寫Ｙ或Ｎ
8. 居留證發證日期 ⇒ 完成月收入確認 / 排入核卡 / 核卡作業，身分證字號第二碼不是 1 或 2 需必填，且西元格式 yyyyMMdd
9. 居留證期限 ⇒ 完成月收入確認 / 排入核卡 / 核卡作業，身分證字號第二碼不是 1 或 2 需必填，且西元格式 yyyyMMdd
10. 居留證背面號碼 ⇒ 完成月收入確認 / 排入核卡 / 核卡作業，身分證字號第二碼不是 1 或 2 需必填
11. 外籍人士指定效期 ⇒ 完成月收入確認 / 排入核卡 / 核卡作業，身分證字號第二碼不是 1 或 2 需必填，且格式 yyyyMM，如果是否永久居留證為 N ，他的居留證期限 = 20270125，外籍人士指定效期 = 202701，如果是否永久居留證為 Y，則外籍人士強制他的居留證期限 = 99991231，外籍人士指定效期 = 999912
12. 舊照查驗 ⇒ 完成月收入確認 / 排入核卡 / 核卡作業，身分證字號第二碼不是 1 或 2 需必填，填寫Ｙ或Ｎ
13. 現職月收入 ⇒ 完成月收入確認 / 排入核卡 / 核卡作業必填
14. 正卡人地址
    - 需要驗證 戶籍地址 、帳單地址、寄卡地址
    - 原卡友驗證完整地址，非卡友切割驗證
    - 上述郵遞區號都要有
15. 附卡人地址
    - 需要驗證寄卡地址
    - 原卡友驗證完整地址，非卡友切割驗證
    - 上述郵遞區號都要有
