### 中風險等級

#### 1. 聯徵查詢階段的產生條件

**觸發時機：** 網通執行聯徵查詢結束後，呼叫某支 SP（預存程序）

**適用對象：** 客戶風險等級為 M（中風險）的案件

**產生條件：** 根據聯徵查詢結果判斷是否產生加強審查執行表

#### 警示帳戶判定規則：

**勾選「是」的情況：**

- 【通報案件紀錄】欄位 = 「Y」
- 【是否有補充註記】欄位 = 「1」或「2」或「A」
- 加強審核執行表產生
  - 客戶基本資料驗證: 聯徵資料勾選，且為警示帳戶

**勾選「否」的情況：**

- 『身分證遺失冒用註記』區塊中：

  - 【通報案件紀錄】欄位 = 「N」
  - 【是否有補充註記】欄位 = 「N」

- 加強審核執行表產生
  - 客戶基本資料驗證: 聯徵資料勾選，且不為警示帳戶
  - 審查意見: 建議核准
  - 經辦簽核: SYSTEM
  - 覆核簽核: SYSTEM
  - 防制洗錢及打擊資恐督導主管簽核: SYSTEM
  - 日期: 押上

#### 2. Paperless 階段的重新產生條件

**觸發時機：** 當風險等級有變動時

**處理方式：** 重新產生「加強審查執行表」
